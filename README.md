## VocationPro


### Introducción
Esta una landing page creada con HTML, CSS y JAVASCRIPT.
El diseño inicial fue creado en Figma, en el proceso se tuvo en cuenta la investigación de mercado de otras plataformas dirigidas al mismo tipo de usuario y con temática en la vocación profesional.
Se inspira en la mejora del proyecto anterior llamado Profesión Vocacional, en el cual solo se crearon paginas en HTML puro (fue uno de mis primero proyectos web).
Este proyecto no tiene licencia como tal y es creado simplemente como una idea para mejorar mis habilidades y adquirir nuevas oportunidades de aprendizaje.


### De que trata:

*	Una página para tener tutorías, orientación profesional y consejos para elegir la profesión más adecuada para ti.
* Si ya sabes que profesión elegir, pero estas en duda con lo que ha escogido, puede tener orientación profesional.
*	Test personalizado para vocación profesional.
